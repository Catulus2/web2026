### Des notions de HTML

1. HTML
HyperText Markup Language (language de balise hypertexte)

=> un fichier html c'est c'est d'abord du texte
=> il est écrit par un `développeur` (avec VScode par exemple)
=> il est vu par un utilisateur (avec FireFox)


Une balise html s'écrit : `<maBalise></maBalise>`

**Ex :** `<h1></h1>` 
Le texte s'écrit **entre les balises.**

**Ex :** `<h1>Mon texte</h1>` 

Il existe aussi des balises orphelines : `<!DOCTYPE html>, <br>, <img>, ...`

Référence: [Mozilla](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements)
